Project Terbaru

Generative RAG AI

This project involves the development of an AI-powered chatbot application using Django, Google Gemini API for generating responses, and Retrieval-Augmented Generation (RAG) with embeddings generated by sentence-transformers/all-MiniLM-L6-v2. The chatbot provides personalized responses by retrieving relevant information from user-provided documents and generating context-aware answers.

House Price Prediction

This final project for the Data Mining course aims to predict house prices using the LightGBM algorithm based on property features. The challenge is derived from the Kaggle competition "House Prices - Advanced Regression Techniques" using data from Ames, Iowa. The main evaluation metric is RMSE between predicted and actual prices. This is a regression project to predict house prices, with SalePrice as the target variable.
